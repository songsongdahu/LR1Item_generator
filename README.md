LR1Item_generator
=========
LR1Item_generator是一个用来生成LR(1)项目集的程序，它以一个LR(1)文法的产生式集合作为输入，经过计算生成LR(1)的项目集以及分析表(parsing tables)。

Class简介:

LR1:包含了first,closure,goto,item等LR(1)项集生成的主要方法

LR1Item:表示一个LR(1)的项，包含一组产生式和一个序号

LR1Pro:表示一条产生式

Symbol:文法符号类，包括终结符，非终结符和ε
